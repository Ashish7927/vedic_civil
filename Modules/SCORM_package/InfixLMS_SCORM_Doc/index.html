SCORM Module Installation

Install SCORM Module In your Existing InfixLMS System

Log into your InfixLMS system and go to System Setting > Module Manager.

Click the Add Module, if you are not added yet.

Browse The module zip file. Then Submit. After Submit you can see You have a verify Button for Verify your module license.

Enter Envato Username & Envato Purchase code and verify your licence

After Successfully verify your licence you can found a enable/disable switch. you can easily enable/disable by click on switch

Install complete

Add Lesson

Add SCORM File into course

Add Scorm into a course is a very easy process. Here are the steps.

  • After Install SCORM Module, Go to Courses From Dashboard Sidebar.
  • Go Action and Click Add Lesson
  • Select Lesson from "ADD CHAPTER / LESSON"
  • You can find SCORM Self in Host Dropdown (Upload file into Self hosting)
  • If You Add InfixLMS AWS s3 Module, Then You can find SCORM AWS S3 Option (Upload file into AWS S3 Hosting)
  • Then Browse SCORM File(Zip Format)
  • Save Lesson. Done
